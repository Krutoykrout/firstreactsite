# AVTO CAPITAL — демо-сайт с админкой

## Как открыть
Откройте `index.html` в браузере или загрузите содержимое папки на GitHub Pages.

## Админка
Адрес: `#/admin`

Логин: `1`
Пароль: `2`

В демо-версии изменения сохраняются в `localStorage` браузера. Это значит, что они остаются на том устройстве и в том браузере, где редактировали сайт.

## Что можно менять
- название компании;
- главный заголовок и тексты;
- телефон, WhatsApp, Telegram, город, адрес, почту, график;
- каталог машин: наличие, статус, цену, характеристики и фото;
- разделы “О компании” и “Отзывы”;
- преимущества и этапы на главной.

## Для настоящей общей админки
Нужен бесплатный backend: Supabase/Firebase/Decap CMS. Тогда изменения будут видны всем посетителям сайта, а не только в браузере администратора.
